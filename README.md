# Pipeliam-Blender
A simple remote based pipeline for Blender (works best on 5.1)


## Introduction 
- What is Pipeliam for Blender ? 
  - A Free open source tool that allows small / independant studios to collaborate by a simple drive access on their projects.

- Why was it created ? 
  - I worked for a few years now in new studios that try desperately for money matters to hop onto blender, and yes it's so powerfull,
  but NO, NO ONE HAS A PIPELINE TOOL FOR IT, and it drives me crazy, 
  So I created mine

- How and who can use it ? 
  - Anybody as long as you credit the Pipeliam in your projects and, if you can, leave a small contribution so that I can continue developping it freely.
